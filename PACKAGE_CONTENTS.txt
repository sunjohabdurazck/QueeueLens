🚀 QUICK START GUIDE
═══════════════════════════════════════════════════════════════

STEP 1: Prerequisites
─────────────────────────────────────────────────────────────
□ Install XAMPP (https://www.apachefriends.org/)
□ Install Composer (https://getcomposer.org/)
□ Create Firebase project
□ Enable Firestore & Authentication

STEP 2: Extract & Install
─────────────────────────────────────────────────────────────
Windows:
  1. Extract to: C:\xampp\htdocs\
  2. Double-click: install.bat
  
Mac/Linux:
  1. Extract to: /Applications/XAMPP/htdocs/
  2. Run: ./install.sh

STEP 3: Firebase Setup
─────────────────────────────────────────────────────────────
1. Download service account key from Firebase Console
2. Save as: config/serviceAccountKey.json
3. Update project ID in config/firebase.php

STEP 4: Create Admin User
─────────────────────────────────────────────────────────────
In Firestore:
  Collection: users
  Document ID: [your Firebase Auth UID]
  Fields: {
    "role": "admin",
    "name": "Your Name",
    "email": "admin@example.com"
  }

STEP 5: Launch!
─────────────────────────────────────────────────────────────
1. Start Apache in XAMPP
2. Visit: http://localhost/admin_system/
3. Login with Firebase credentials
4. Enjoy! 🎉
